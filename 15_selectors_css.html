CSS Selectors are used to find element whose property will be set or used to target to html elements. Selectors make it easy for us to easily target single or multiple HTML elements to markup.

Four types of selectors in CSS are :-

1. CSS element selector
2. CSS id selector
3. CSS class selector
4. CSS grouping selector
